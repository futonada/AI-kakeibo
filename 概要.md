## 概要
Claude sonnet 4.6を用いて画像解析とチャット分析を可能にしたiOS家計簿アプリ。
手入力で家計簿を付けたり、分析するのが面倒で続かない人にも使えるよう、
レシートを撮影するだけで家計簿が付けられ、分析もAIに丸投げ可能な仕様にして解決。

## 全体構成
iOSアプリ（SwiftUI + SwiftData）
  ↓ レシート画像 multipart/form-data
バックエンドAPI（Python + FastAPI on Render）
  ↓ base64エンコード画像
Claude Sonnet 4.6 via LangChain（with_structured_output）
  ↓ 構造化JSON返却
iOSアプリ → SwiftDataにローカル保存

## レイヤー構成
iOS UI	SwiftUI（ネイティブ）	4タブ構成
iOS DB	SwiftData	ローカルのみ
画像選択	PhotosUI	フォトライブラリ（カメラは未対応）
API通信	APIClient.swift	multipart/form-data
バックエンド	FastAPI + uvicorn	Renderにデプロイ済み
LLM連携	LangChain with_structured_output	PydanticOutputParserは不採用
LLMモデル	claude-sonnet-4-6	Vision対応


## iOSの画面構成（4タブ）
ホーム — 月別集計・カテゴリ別棒/円グラフ・来店ランキング（Swift Charts）
追加 — フォトライブラリで画像選択 → API送信 → 確認・編集 → 保存
履歴 — レシート一覧・詳細・編集・削除
AI相談 — 月のレシートデータをコンテキストとしてClaudeとチャット

## バックエンドのエンドポイント
POST /analyze-receipt — レシート画像解析 → 構造化JSON返却
POST /chat — レシートデータを埋め込んだAI相談
